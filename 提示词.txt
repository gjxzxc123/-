 1. 终极版“超级提示词” (One-Shot Prompt)
如果你希望一次性描述出这个程序的核心逻辑，可以使用这段 Prompt：
角色设定：请你作为一名资深的前端图形学工程师，使用 HTML5、Three.js 和 MediaPipe Hands 库。
项目目标：开发一个 “3D 梦幻互动音乐圣诞树” 的单页 Web 应用。
核心功能要求：
3D 渲染：使用粒子系统构建螺旋状的圣诞树，粒子包含树叶、装饰球和发光点。使用自定义 Shader (GLSL) 实现粒子从“树形”到“球形爆炸”的平滑变形动画。场景中需包含 3D 旋转五角星（随树高度移动）和下落的雪花系统。
音频可视化：使用 Web Audio API 分析音频频域（AnalyserNode），获取低频（Beat）强度，让树上的装饰球随音乐节奏闪烁、跳动，且树整体随节奏轻微膨胀。需解决浏览器自动播放限制（添加启动页）。
AI 手势交互 (分工明确)：
左手 (Left Hand)：计算手掌张开程度，控制树的“爆炸/聚合”程度。手张开时，树和挂在树上的照片向外炸开；握拳时还原。
右手 (Right Hand)：计算拇指与食指的距离。当双指捏合时，屏幕中央随机弹出一张用户上传的照片；松开时照片消失。
照片系统：允许用户批量上传照片。照片需裁剪为圆形纹理，随机挂载在树枝内部，并跟随树的爆炸效果同步位移。
UI 控制面板：提供全屏切换、背景风格切换（纯黑/极光/暖冬等）、音乐上传及播放控制、以及参数调节（树的高度、旋转速度、粒子大小、音乐律动灵敏度）。
代码结构：单文件 HTML，代码需健壮，处理好 Canvas 纹理生成和 MediaPipe 的异步加载。
